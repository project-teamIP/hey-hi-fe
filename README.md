# 헤이, 안녕 🖐

요즘 핫한 외국어 공부, 다른 사람과 함께 하고 싶다면? 헤이 안녕을 사용해주세요. 헤이 안녕은 한영권 사람들이 1대 1 영상통화를 통해 서로의 언어를 나누는 언어교환 서비스 입니다.
